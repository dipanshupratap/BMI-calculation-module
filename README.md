#  Simple BMI Calculator + Progress Tracker (Beginner Version)

A beginner-friendly **Python terminal project** that helps you calculate your **Body Mass Index (BMI)** and track your **fitness progress** (calories or steps) over time — right from your terminal.

---

# Features

✅ **BMI Calculator**
- Calculates BMI using weight (kg) and height (m)
- Displays BMI category:
  - Underweight
  - Normal weight
  - Overweight
  - Obese

✅ **Progress Comparison Tool**
- Track Calories or Steps
- Compare progress (Daily / Weekly / Monthly)
- Shows average and improvement messages

✅ **Menu-Based Interface**
- Simple text-based interaction
- Easy for beginners to understand and modify
